# Python Cheatsheet

#### Python 2 vs Python 3[^1]

| Python2 | Python3 |
|---|---|
| 5 / 2 = 2  | 5 / 2 = 2.5 | 
| strings = ascii | string = unicode | 
| `except Exception, e:` | `except Exception as e` | 
| `raw_input()` | `input()` | 
| `long`/`int` type | `int` type | 
| `xrange` | `range()` |
| `print x` | `print(x)` |
